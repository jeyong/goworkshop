## Go 부트캠프

이 자료는 다른 프로그래밍 언어에 대해서 경험이 있고 Go를 배우려는 초/중급 개발자를 위한 자료입니다. Go를 시작하거나 자세히 Go와 내부를 이해하고자 하는 개발자에게 적합합니다.
